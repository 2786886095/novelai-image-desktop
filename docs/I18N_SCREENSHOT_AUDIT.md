## 漂移/不整齐判定标准

- 文本不可被裁切；单行按钮文字太长时应换行、缩小或保持按钮自适应。
- 同一行按钮高度一致；同组卡片左边距、右边距、圆角与间距一致。
- 导航项在目标语言下不应覆盖图标；手机底栏最多显示 3 个主入口 + 更多。
- 表单 label 与 input/select/textarea 不应错位；多语言长词不应压缩到不可读。
- 弹窗底部按钮在窄屏下不应越界；必要时换成纵向布局。
- 截图中发现问题时，记录：语言、端、页面、视口、截图路径、问题等级、修复文件。
