## Langbai NovelAI Studio

请根据你的操作系统下载对应安装包：

| 系统 | 安装包 | 安装说明 |
| --- | --- | --- |
| 🪟 **Windows** (x64) | `Langbai-NovelAI-Studio-1.2.6.exe` | 便携版，双击即用，无需安装 |
| 🍎 **macOS** (Intel + Apple 芯片通用) | `Langbai-NovelAI-Studio-1.2.6-universal.dmg` | 拖入「应用程序」；**未签名**，首次打开请右键 →「打开」 |

### v1.2.6 更新内容

- **转换/反推历史记录改为点击展开**：上一版历史记录会把提示词内容（一版或两版）直接铺开显示，条数一多列表就很长；现在每条记录默认折叠，只显示当时输入的描述，点击该条记录才会展开——已知角色模式生成的记录展开后是角色名版/特征版两张卡片，普通记录展开后是一版提示词，各自都能单独复用。

### v1.2.5 更新内容

- **修复已知角色模式下转换/反推记录只显示一版提示词的问题**：查看历史记录时，之前无论是否开启了"同时生成角色名版/特征版"，都只展示合并后的单一提示词；现已改为跟当次生成结果一样，历史记录里也会分别展示两个版本，并且每个版本都能单独复用。

### v1.2.4 更新内容

- **修复移动端反推/转换"开了双版本却总是只出一个"的问题**：已知角色模式下要求同时生成角色名版/特征版两个版本的指令，之前只写进了移动端的系统提示词，用户消息里完全没有；桌面端是系统提示词和用户消息里都写了（双重强化）。现已把移动端补齐到跟桌面端一致，双版本生成恢复正常。
- **修复移动端精准参考图默认参数错误（可能一直有画质问题）**：新加一张精准参考图时，移动端默认类型是"角色+画风"，桌面端早在更早版本就改成了只用"角色"，因为"角色+画风"会导致纹理/半调网点伪影；移动端当时移植时抄的是桌面端修复前的旧默认值，现已改正。同时修复了图片尺寸与官方比例不符需要补边时，移动端一直补黑色（桌面端补白色，黑色补边本身就是网点伪影的来源之一）的问题。
- **修复移动端"表情迁移"等后期处理工具发送畸形参数**：强度/表情等级之前当作小数处理，实际发给 API 的是"happy;;3.0"这种带小数点的字符串，桌面端发的是干净的"3"；现已改成取整数。
- **修复移动端固定种子默认值**：切换到"固定种子"且未手动填写时，之前固定给种子 1，现已改成和桌面端一样随机滚一个。
- **转换/反推队列成功后自动消失**：任务完成后 1.5 秒自动从任务列表移除，无需再手动点击删除；失败的任务仍会保留，方便查看失败原因（结果和历史记录不受影响）。
- **新增"跨次启动记住参数"设置**：桌面端设置 → 性能、移动端设置页新增独立开关，分别控制文生图、图生图（仅桌面端）、局部重绘、超分、后期处理是否记住上次使用的参数，默认全部开启，不影响现有使用习惯。

### v1.2.3 更新内容

- **反推与转换的中文提示词修复对齐**：上一版修复的"已知角色模式系统提示词中英混杂导致质量下降"问题，确认反推（AI 反推）与转换共用同一套逻辑，已同步验证生效；移动端此前遗漏了这次中文化修复，现已补齐，反推/转换不再是先出一版粗糙结果再返回第二版的旧模式。
- **转换/反推改为后台并发处理，新增任务队列与历史记录（桌面端 + 移动端）**：点击「开始转换/反推」后可立即发起下一次请求，无需等待上一次完成；新增默认折叠的任务列表，实时显示每条请求的处理中/完成/失败状态。同时新增独立的转换记录与反推记录页面，可查看所有历史结果；反推记录会在原图被删除后自动清理，也支持手动清空。
- **任务队列新增取消功能**：处理中的任务现在也能点击移除，移除后会立刻从列表消失，且该请求即使之后返回结果也不会再显示或写入历史记录。
- **历史记录改为显示输入描述，而不是生成结果**：转换/反推的历史列表之前展示的是 AI 生成出来的提示词，现在改为展示你当初输入的描述文字，更方便根据记忆找到对应记录（无输入文本的反推记录会回退显示生成结果）。
- **修复桌面端队列/历史记录面板颜色对比度过低看不清的问题**：面板文字之前误用了从未定义过的颜色变量，一直依赖深色主题的兜底色显示，在实际的浅色工作台主题下几乎看不清；现已改用正确定义的文字颜色变量。
- **反推与转换统一为单次请求策略**：已知角色模式下，如果 AI 第一次没有返回完整的角色名版/特征版两个版本，之前会自动发起第二次"修复"请求；现在改为直接采用第一次的结果，反推和转换都保证只请求一次，不再出现有时一次、有时两次的情况。
- **重绘的正面提示词不再跟生图共享**：之前重绘页面的正面提示词和文生图/图生图是同一个输入框，改动生图的提示词会连带影响重绘；现在重绘拥有独立的正面提示词，互不干扰（负面提示词、风格词等其余参数仍与生图共享，未改动）。
- **修复移动端重绘蒙版效果差、前后对比看不出区别的问题**：移动端画的蒙版之前是平滑自由笔迹直接上传，NovelAI 的重绘接口是按 64 像素网格处理蒙版的，笔迹如果没对齐网格容易导致重绘范围小、效果弱甚至看不出变化；现已改成和桌面端一致的做法——画的时候仍然是自由笔刷手感，导出时自动扩展对齐到完整的 64×64 网格，重绘效果与前后对比恢复正常。
- **重绘完成后自动清除画布上的旧蒙版笔迹**，避免新结果图上还叠着上一次画的蒙版痕迹。

### v1.2.2 更新内容
